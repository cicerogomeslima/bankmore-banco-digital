# BankMore.E2E.Tests

Testes via Postman.

## Como rodar os testes via Postman

1) Importar os arquivos descritos abaixo no Postman

```bash
bankmore_postman_environment.json
bankmore_postman_collection.json
```

2) Rodar os testes dentro do Postman.


## Como rodar os testes via curl
```
Rodar via bash
chmod +x bankmore_e2e_curl.sh
./bankmore_e2e_curl.sh
```

```
Rodar via PowerShell do Windows
Set-ExecutionPolicy -Scope Process -ExecutionPolicy Bypass
.\bankmore_e2e_curl.ps1
```
